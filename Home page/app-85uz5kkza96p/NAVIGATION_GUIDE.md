## ðŸŽ¯ Quick Access Guide

### To Access Spectral Analysis (FFT, Wavelet, EOF/PCA)
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Spectral" tab
3. Adjust FFT window size slider
4. View power spectrum, wavelet coefficients, and EOF results
```

### To Access Geostatistical Analysis (Kriging, Variogram)
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Geostatistics" tab
3. Select variogram model (Spherical, Exponential, etc.)
4. View experimental variogram and kriging parameters
```

### To Access Extreme Value Analysis
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Extreme Values" tab
3. View return period plot and block maxima distribution
4. See return levels for 2, 5, 10, 20, 50, 100-year events
```

### To Access Ensemble Analysis
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Ensemble" tab
3. Adjust number of ensemble members (3-20)
4. View ensemble mean, spread, and uncertainty bands
```

### To Access Machine Learning Features
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Machine Learning" tab
3. Select model type (PINN, Super-Resolution, Transfer Learning, Anomaly Attribution)
4. View model metrics and performance statistics
```

### To Access Anomaly Detection
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Anomaly" tab
3. View detected anomalies with Z-score threshold
4. See attribution categories (Extreme High, High, Low, Extreme Low)
```

### To Access Climate Indices
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Climate Indices" tab
3. View anomaly index, standardized index, and trend analysis
4. See trend significance and warming/cooling direction
```

### To Access Advanced Integration Features
```
1. Click "Advanced Analysis" in navigation menu
2. Click "Advanced" tab
3. View available integration options:
   - Numerical Model Coupling (WRF, ROMS, HYCOM)
   - Real-Time Sensor Networks
   - Trajectory Analysis
   - Ocean Mixed Layer Analysis
```
